Учебные примеры
***************

.. contents::
    :local:
    :depth: 2

Мы предоставляем несколько обучающих примеров, демонстрирующих базовое
использование GNN-AID. Исходный код примеров находится в каталоге
``tutorials/``.

01 - базовая функциональность
---------------------------------------------------

Учебный пример 0 посвящён базовой функциональности и включает:

1. Загрузку датасета и подготовку данных.
2. Построение модели GNN.
3. Обучение модели GNN.
4. Сохранение и загрузку весов модели.
5. Оценку качества обученной модели.

Следующие учебные примеры **01–05** демонстрируют различные сценарии
использования фреймворка.

01 - атака уклонения (evasion)
---------------------------------------------------

Демонстрирует поведение графовой модели при атаке уклонения
(*evasion attack*), а также защитные возможности защитника на основе
градиентной регуляризации (*Gradient Regularization Defender*).
Используются датасет **Cora** и модель **GNN**.

02 - атака отравления (poison)
---------------------------------------------------

Демонстрирует поведение графовой модели при атаке отравления данных
(*poisoning attack*), а также защитные возможности защитника на основе
коэффициента Жаккара (*Jaccard-based defender*).
Используются датасет **Cora** и модель **GIN**.

.. note::
   Добавить ссылку на файл учебного примера.

03 - атака вывода принадлежности (MI)
---------------------------------------------------

Демонстрирует поведение графовой модели при атаке на вывод принадлежности
(*membership inference attack*) и эффективность защиты на основе
добавления шума к выходу модели (*output noise-based defense*).
Используются датасет **Cora** и модель **GCN**.

.. note::
   Добавить ссылку на файл учебного примера.

04 - интерпретация GNN
---------------------------------------------------

Демонстрирует, как интерпретировать поведение обученной графовой нейронной
сети (GNN) с использованием встроенного интерпретатора
**GNNExplainer**.

GNNExplainer позволяет определить, какие вершины, признаки и связи
повлияли на процесс принятия решений моделью. Фреймворк GNN-AID
поддерживает различные интерпретаторы; в данном примере используется
GNNExplainer. При необходимости вы можете легко изменить интерпретатор
в конфигурации.

.. note::
   Добавить ссылку на файл учебного примера.

05 - комбинирование нескольких атак, защит
---------------------------------------------------

Демонстрирует сложные сценарии использования фреймворка, включая
комбинированные атаки, многоуровневые пайплайны защиты и интегрированные
рабочие процессы интерпретации.
Используются датасет **Cora** и модель **GIN**.
